## 💫 Features

- 🧠 AI Chat & Commands (ChatGPT / Gemini)
- 🎭 Reaction Plugins (.slap, .hug, .bully, etc)
- 🧷 Group Moderation (.antilink, .ban, .warn)
- 📥 Downloader (.play, .ytmp3, .ytmp4)
- 🎮 Fun & Games (.tictactoe, .roast, .findwife)
- 🌐 Public / Private Mode
- 🎀 Cute and dark styled menus
